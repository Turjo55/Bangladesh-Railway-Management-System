# Bangladesh Railway Management System (Node.js Edition)

This is the full rewrite of the system using **Node.js, Express, and MongoDB**.

Since you do not have Node.js installed, follows these steps FIRST:
1.  Go to [nodejs.org](https://nodejs.org/).
2.  Download the **LTS Version** (Recommended for Most Users).
3.  Install it.
4.  Open a **NEW** terminal/command prompt/PowerShell.
5.  Check if it's installed by typing `node -v`.

## ðŸš€ Setup & Run

Once Node.js is installed:

### 1. Open Terminal in Project Folder
Navigate to: `d:\Xampp and all\XAMPP\htdocs\Bangladesh Railway Management System`

### 2. Install Dependencies
Run this command to download all required libraries:
```bash
npm install
```

### 3. Initialize Database
Run this command once to reset and populate the database:
```bash
npm run seed
```

### 4. Start the Server
Run:
```bash
npm start
```
You will see: `Server running on http://localhost:3000`

### 5. Open Browser
Go to **[http://localhost:3000](http://localhost:3000)**
